Makko.AI Collection Export

Contents
  Each collection folder mirrors your Makko.AI collection tree.
  - "Concept art/"        concept images for the collection
  - "{Character} Sprites/"    sprite sheets + metadata JSON for one character
  - "{Character} References/" reference images for one character
  - "Backgrounds/"        background art linked to this collection
  - "Props/"              prop art linked to this collection
  Sub-collections are nested as subfolders. Concept art and reference
  images are omitted when "Game assets only" is selected at export time.
